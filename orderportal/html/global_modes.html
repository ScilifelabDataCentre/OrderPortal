These settings control the behaviour of the portal on a global scale. If the portal needs to be partially shut down for some reason, then set these modes accordingly.

The changes take effect immediately, so use with care. For example, if order editing is shut off, then a user currently editing an order, will no longer be able to save the order. Therefore, change settings during non-working hours, e.g. early morning, late afternoon, or holidays.
