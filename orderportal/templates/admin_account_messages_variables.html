{# Table of descriptions of variables to use in account messages. #}
Variables available for use in the subject and text content of account messages.
Variable Value
{site} The name of the OrderPortal site.
{support} The contact email for the OrderPortal site.
{account} The email of the user account.
{url} The URL of the used account.
{password_url} The URL for the password setting page, if relevant.
{password_code_url} The URL for the password setting page, including code and user account email, if relevant.
{code} The password setting code, if relevent.
