The settings are defined in the YAML settings file given by SETTINGS_FILE (see below), which is read on startup of the OrderPortal server.

To change the settings, the YAML file must be edited and the OrderPortal server restarted. Only the sysadmin for the server machine can do this.

The full list of configuration variables is in the OrderPortal/orderportal/__init__.py file.
